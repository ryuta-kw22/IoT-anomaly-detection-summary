# IoT Device Anomaly Detection via Power waveform Analysis
（日本語タイトル：消費電力波形解析に基づくIoTデバイスの異常動作検知手法に関する研究）

## Overview (概要)
本リポジトリは、IoTデバイスの物理的な消費電力波形を用いた異常動作検知手法の研究のうち、「文法ベースの手法（「長期的な遷移」を考慮した「説明可能な」異常動作検知手法）」に関する研究のサマリーとなります。
※本研究は早稲田大学 戸川研究室（知的財産）に帰属するため、コアとなるソースコードやデータセットは非公開としています。
本ページでは研究のアーキテクチャとアプローチの概要を解説します。

## Motivation & Problem (背景と課題)
### 1. サプライチェーンの複雑化とセキュリティリスク
IoTデバイスの急速な普及に伴い、ICチップ製造の低コスト化・効率化を目的とした製造の外注化・分業化が進んでいます。しかし、この構造はサプライチェーンを複雑化させ、深刻なセキュリティリスクを生み出しています。

*   **製造時の脅威:** 悪意ある第三者による「ハードウェアトロイ」の挿入。
*   **出荷後の脅威:** マルウェア感染によるソフトウェア層への攻撃。

これらの攻撃は、機能停止や内部情報の漏洩といった異常動作を引き起こし、デバイスの信頼性を著しく低下させます。設計情報の完全な入手が困難な現代の製造環境において、デバイスが製造・出荷された後も継続的に監視し、異常動作を包括的に検知する技術が求められています。

### 2. サイドチャネル解析による異常検知
出荷後のリアルタイムな検知手法として、デバイスから漏洩する物理情報（サイドチャネル情報）を分析する手法が注目されています。特に**消費電力**は、IoTデバイスから比較的容易かつ低コストに取得できるため、継続的な監視に適したデータソースです。

### 3. 従来手法における課題
消費電力を用いた既存の異常検知手法（深層学習や形状比較による解析）には、実用化に向けた大きな壁が存在します。

*   **判定根拠の「ブラックボックス化」**
    深層学習を用いた手法は高い検知精度を誇る一方で、「なぜ異常と判定されたのか」という根拠が不明確です。セキュリティ運用において、判定プロセスの説明責任（Accountability）を果たせないことは、信頼性の観点から大きな課題となります。
*   **文脈的異常（Contextual Anomaly）の看過**
    多くの手法は「局所的な波形の特徴」に着目しています。しかし、IoTデバイスの異常は、瞬時的な電力値の逸脱よりも、正常な動作順序や遷移からの逸脱（スキップ等）として現れる場合が多くあります。波形の形そのものが正常に見える場合、従来の「形状比較」や「短期的な遷移解析」では、長期的な文脈上の異常を検知できません。

### 4. 本プロジェクトの目的
本研究では、これらの課題を解決するために「**物理的な波形データを論理的な記号列へと変換し、言語的に解析する**」というアプローチを提案します。
これにより、複雑な動作遷移を**構造的**に捉え、「低負荷・高精度」かつ「判定根拠が明確（説明可能）」なセキュリティ基盤の実現を目指しています。

## Methodology (提案手法：AE-GIアプローチ)
これらの課題を解決するため、以下の2段階のアプローチ（AE-GI手法）を構築しました。

1. **特徴抽出と記号化 (Autoencoder & Clustering)**
   - デバイスの消費電力波形を時系列データとして取得。
   - Autoencoder（AE）とクラスタリング（MeanShift法）を用い、波形の幾何学的特徴を「記号（アルファベット等）」に変換。物理データを論理データへマッピングします。
2. **文法推論による文脈異常検知 (Grammatical Inference)**
   - 記号化されたシーケンスに対し、文法推論を適用して正常な状態遷移モデル（オートマトン）を構築。
   - 遷移のルールから外れた動き（構造的な異常）を検知することで、「どの時点の、どの動作遷移がおかしいか」という判定根拠を明確に説明可能にしました。

## Key Results (成果)
- 従来手法と比較して、検知精度を **1.2倍** 向上させることに成功。(より多くの従来手法に対して定量的な評価を進めている)
- 深層学習手法と提案手法の位置付けの明確化。（提案手法は深層学習+文法解析のハイブリッド手法）
- 判定根拠の可視化により、セキュリティ運用における実用性・解釈性を大幅に改善。
